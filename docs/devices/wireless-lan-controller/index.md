# Wireless LAN Controller

Controllers that manage enterprise Wi-Fi access points, either on-prem
appliances or cloud-hosted control planes.

## Selection criteria

- Vendor positioned as a Leader in the Gartner Magic Quadrant for
  Enterprise Wired and Wireless LAN Infrastructure.
- Product line shipping and updated within the last 3 years.
- Manages enterprise APs (campus, branch, warehouse) — not carrier Wi-Fi
  or ISP hotspots.
