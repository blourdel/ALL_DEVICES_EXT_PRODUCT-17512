# SD-WAN

Software-Defined WAN edge appliances and their controllers /
orchestrators, used to connect enterprise branches to data centers and
cloud.

## Selection criteria

- Vendor positioned as a Leader in the Gartner Magic Quadrant for
  SD-WAN.
- Product line shipping and updated within the last 3 years.
- Enterprise branch / multi-cloud connectivity (not ISP managed-WAN
  backbones).

## In-scope vendors and product lines

| Vendor             | Recent product lines                                       |
|--------------------|------------------------------------------------------------|
| Cisco              | Catalyst SD-WAN (Viptela) on Catalyst 8000, Meraki MX      |
| Fortinet           | Secure SD-WAN on FortiGate, FortiManager orchestration     |
| Palo Alto Networks | Prisma SD-WAN (CloudGenius ION appliances)                 |
| HPE Aruba          | EdgeConnect SD-WAN (Silver Peak), Orchestrator             |
| Versa Networks     | Versa Operating System (VOS) appliances, Versa Director    |
| VMware / Broadcom  | VeloCloud SD-WAN Edge                                      |
| Juniper Networks   | Session Smart Router (128 Technology), Mist WAN Assurance  |

## Out of scope

- ISP managed-WAN gateways without enterprise-managed control plane.
