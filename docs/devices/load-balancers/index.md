# Load Balancers

Application Delivery Controllers (ADC) and L4/L7 load balancers used by
enterprises in data centers and clouds.

## Selection criteria

- Vendor recognized as a Leader in Gartner's evaluations for Application
  Delivery Controllers / Application Security & Delivery.
- Product line shipping and updated within the last 3 years.
- Enterprise / data-center / multi-cloud usage.

## In-scope vendors and product lines

| Vendor                       | Recent product lines                       |
|------------------------------|--------------------------------------------|
| F5                           | BIG-IP iSeries / rSeries, BIG-IP Next, NGINX Plus, Distributed Cloud |
| Citrix / Cloud Software Group| NetScaler ADC (MPX, SDX, VPX, BLX, CPX)    |
| A10 Networks                 | Thunder ADC                                |
| Radware                      | Alteon                                     |
| Broadcom                     | Avi Load Balancer (ex-VMware NSX ALB)      |
| Kemp / Progress              | LoadMaster                                 |
| Array Networks               | APV Series                                 |

## Out of scope

- Pure cloud-provider managed LBs (AWS ELB, Azure LB, GCP LB) — covered
  by their respective cloud-native extensions, not network-device ones.
