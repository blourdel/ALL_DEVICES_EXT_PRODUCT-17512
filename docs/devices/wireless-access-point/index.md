# Wireless Access Point

Enterprise Wi-Fi access points (Wi-Fi 6 / 6E / 7), both indoor and
outdoor models managed by the controllers listed in
[wireless-lan-controller](../wireless-lan-controller/index.md).

## Selection criteria

- Same vendor set as the Wireless LAN Controller list (Gartner Leaders in
  Enterprise Wired and Wireless LAN Infrastructure).
- AP family shipping and updated within the last 3 years; preference for
  Wi-Fi 6E / Wi-Fi 7 capable models.
- Enterprise deployment (campus, branch, warehouse, healthcare,
  industrial) — not consumer or ISP-CPE Wi-Fi.

## Out of scope

- Consumer-grade APs and mesh kits (eero, Google Nest, ASUS ZenWifi).
- ISP-issued residential gateways with embedded Wi-Fi.
