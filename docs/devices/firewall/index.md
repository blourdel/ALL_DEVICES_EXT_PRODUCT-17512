# Firewall

Next-generation firewalls (NGFW) deployed at the perimeter, in the data
center, or as virtual / cloud instances.

## Selection criteria

- Vendor positioned as a Leader in the Gartner Magic Quadrant for
  Network Firewalls.
- Product line actively sold and updated within the last 3 years.
- Enterprise targeting (physical, virtual, or cloud NGFW).

## In-scope vendors and product lines

| Vendor                | Recent product lines                              |
|-----------------------|---------------------------------------------------|
| Palo Alto Networks    | PA-Series (PA-400 / 1400 / 3400 / 5400 / 7000), VM-Series, CN-Series |
| Fortinet              | FortiGate (40F – 7000F), FortiGate-VM             |
| Check Point           | Quantum Force, Quantum Spark, CloudGuard Network  |
| Cisco                 | Secure Firewall 1000 / 3100 / 4200 / 9300 (Firepower / FTD) |
| Juniper Networks      | SRX Series (SRX300 – SRX5000), vSRX, cSRX         |
| Sophos                | XGS Series, Sophos Firewall (virtual / cloud)     |
| SonicWall             | TZ, NSa, NSsp Series                              |

## Out of scope

- Pure host / endpoint firewalls.
- Carrier-grade DDoS scrubbing appliances.
